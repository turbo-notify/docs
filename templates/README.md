# Documentation Templates

> Reusable templates for consistent documentation.

---

## Available Templates

| Template | Purpose |
|----------|---------|
| [Architecture Document](architecture-template.md) | System design documentation |
| [Runbook](runbook-template.md) | Operational procedure |
| [Guide](guide-template.md) | Tutorial or how-to |

---

## Usage

1. Copy the template file
2. Rename to match your document
3. Fill in the sections
4. Remove unused sections
5. Update links and references

---

## Creating New Templates

When creating a new template:

1. Use consistent structure
2. Include placeholder text
3. Add comments for guidance
4. Document the template here
