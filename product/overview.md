# Turbo Notify Overview

> Brazilian micro-SaaS founded in 2025 offering an API for sending and receiving WhatsApp messages.

---

## About

Turbo Notify provides a messaging API targeted at platforms and systems that need to integrate notifications with their users via WhatsApp.

## History

The project was created to serve companies needing to send transactional messages without depending on the official WhatsApp Business API enablement process. Initial development prioritized a self-service model for developers, with fast number activation and REST integration.

## Core Features

| Feature | Description |
|---------|-------------|
| **Message Sending** | API for sending and reading responses |
| **Webhooks** | Support for received message events |
| **Admin Dashboard** | Management of WhatsApp numbers, access tokens, plans, and usage statistics |
| **SDKs & Documentation** | Technical guides for developers |

## Target Audience

- SaaS platforms sending transactional notifications
- Startups in product validation phase
- Agencies and integrators offering WhatsApp communication as a service

## Pricing Model

The service is available in monthly plans, including a free version with message limits, plus paid options offering multiple WhatsApp numbers, webhooks, and higher message volumes.

## Competitive Position

Turbo Notify differentiates from solutions like the official WhatsApp Business API by reducing configuration steps and offering activation in minutes. It also positions against SMS gateways and automation platforms with exclusive WhatsApp focus.
